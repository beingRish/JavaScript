Regex Methods

test() : Returns a Boolean value that indicates whether or not a pattern exists in a searched string.

exec() : Executes a search on a string using a regular expression pattern, and returns an array containing the results of that search.

String Methods

match() : Matches a string or an object that supports being matched against, and returns an array containing the results of that search, or null if no matches are found.

Works same as exec()

search() : Finds the first substring match in a regular expression search and returns the index.

replace() : An object that supports searching for and replacing matches within a string.

split() : Split a string into substrings using the specified separator and return them as an array.
